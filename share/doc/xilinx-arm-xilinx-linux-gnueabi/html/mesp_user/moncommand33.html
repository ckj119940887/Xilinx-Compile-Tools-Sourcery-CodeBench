LN Load Names

Availability: ALL

Syntax

LN[A|O] [filename] ...

Description

The Load Names command reloads symbols for the current program into MON's symbol table, or loads symbols from the specified files. The new symbols will replace any existing symbols by default, or they can be added to the existing symbols.

  • LN, LNO — Overwrite the existing symbol table.
  • LNA — Add to the existing symbol table.
  • filename — The name of an executable file whose symbols are to be loaded.
