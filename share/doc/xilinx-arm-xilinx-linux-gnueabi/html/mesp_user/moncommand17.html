DS Display Scripts

Availability: ALL

Syntax

DS[Q|V] [file_spec]

Description

The Display Scripts command displays the name and call entry points of any MEP script files found on the current search path. In verbose mode, each directory searched is identified/displayed. The default script files extension is .maj and if no file extension is given then only .maj files are shown, otherwise only files matching the given extension are shown.

  • Q - means Quiet mode.
  • V - means Verbose mode.
  • file_spec has the form : { A-Z | a-z | 0-9 | _ | . | * }...
