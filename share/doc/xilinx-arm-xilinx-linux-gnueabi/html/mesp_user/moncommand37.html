mmu_xlate

Availability: ALL

Note

This is implemented as a script file. It is not a built-in MON command. See Command Script Files.

 

Syntax

mmu_xlate address

Description

The mmu_xlate script translates a virtual address into the corresponding physical address and displays the result. For ARM processors, the translation is based on the mappings to the memory resident MMU map to which the TBASE register points. For MIPS processors, the translation is based on mappings programmed in the TLBs plus the hard-mapped kernel segments.

Example

For ARM: 

mmu_xlate @r0   /* translate address in .r0 */



Virtual  Address      :  01000000

Physical Address      :  00000000

For MIPS: 

mmu_xlate @t0   /* translate address in .t0 */



Virtual Address:   00555050

Physical Address:  00020050
