BC Breakpoint Clear

Availability: ALL

Syntax

BC [*]

BC {#number |addr}

Description

The Breakpoint Clear command clears one or more software breakpoints previously set with the BS command. To temporarily disable a breakpoint, use the -B command instead.

  • *— All software breakpoints are cleared.
  • # number— Specifies the number, as shown by the BL command, of a software or hardware breakpoint to be cleared.
  • addr — Specifies the address of a software breakpoint to be cleared. addr must be an address in a valid code address space. The addr syntax is shown in addr Address.

If no parameter is given and there is a software breakpoint set at the current execute location, that breakpoint is cleared.

Examples 

BC

BC *

BC my_sym
